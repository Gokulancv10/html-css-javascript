CSS Selectors

CSS Selectors are used to find/select the elements that we want to style.

1. Simple Selector: Selects elements based on name, id, class.

I am simple selector example-1
I am simple selector example-2

2. Combination Selector: Select elements based on a specific relationship between them.

i, Child Selector [>]: Select all the elements that are children to the specified element.


ii, Adjacent Sibling Selector [+]: Select an element that directly after another specific element.


iii, General Sibling Selector [~]: Selects all the elements after the end tag of the element.


3. Pseudo Selector: Select elements based on a certain state.

i, Anchor Pseudo Selector: Style applies on anchor tags.


4. Attribute Selector: Select element based on attribute or attribute value.
